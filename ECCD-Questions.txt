Static (4):
1.	Points to a family member when asked to do so - Static (requires real person)
2.	Points to five body parts on himself when asked to do so - Static (requires physical self)
3.	Follows one-step instructions that include simple prepositions (e.g., in, on, under, etc.) - Static (requires physical objects/space)
4.	Follows two-step instructions that include simple prepositions - Static (requires physical objects/space)

EXPRESSIVE LANGUAGE DOMAIN (8 items)
Static (7):
1.	Uses five to 20 recognizable words - Static (parental report)
2.	Uses pronouns (e.g. I, me, ako, akin) - Static (parental report)
3.	Uses two- to three-word verb-noun combinations (e.g., hingi gatas) - Static (parental report)
4.	Speaks in grammatically correct two- to three-word sentences - Static (parental report)
5.	Asks "what" questions - Static (parental report)
6.	Asks "who" and "why" questions - Static (parental report)
7.	Gives account of recent experiences (with prompting) in order of occurrence using past tense - Static (parental report)
Interactive (1): 
4. Names objects in pictures - Interactive (show images; child names verbally or types/selects from choices)

COGNITIVE DOMAIN (21 items)
Static (12):
1.	Looks in the direction of fallen object - Static (requires physical object)
2.	Looks for a partially hidden object - Static (requires physical object/towel)
3.	Imitates behavior just seen a few minutes earlier - Static (parental report)
4.	Offers an object but will not release it - Static (parental report)
5.	Looks for a completely hidden object - Static (requires physical object/towel)
6.	Exhibits simple "pretend" play (feeds, put doll to sleep) - Static (requires physical props)
7.	Matches objects - Static (requires physical objects: spoons, balls, blocks)
8.	Sorts based on shapes - Static (requires physical shapes)
9.	Sorts objects based on two attributes (e.g., size and color) - Static (requires physical shapes)
10.	Arranges objects according to size from smallest to biggest - Static (requires physical graduated shapes)
11.	Copies shapes - Static (requires paper/pen for physical drawing)
12.	Can assemble simple puzzles - Static (requires physical puzzle pieces)
Interactive (9): 
8. Matches two to three colors - Interactive (can present color tiles on screen; child clicks matching pairs) 
9. Matches pictures - Interactive (show picture pairs on screen; child matches by clicking/dragging) 
13. Names four to six colors - Interactive (show colored objects/swatches; child names or selects color name) 
15. Names 3 animals or vegetables when asked - Interactive (show images; child names verbally or selects from choices) 
16. States what common household items are used for - Interactive (show images of household items; child selects or states function) 
18. Demonstrates an understanding of opposites by completing a statement - Interactive (present statement with images; child selects opposite) 
19. Points to left and right sides of body - Interactive (show body diagram; child clicks left/right parts as instructed - though physical is more accurate) 
20. Can state what is silly or wrong with pictures - Interactive (show pictures with errors; child identifies/clicks what's wrong) 
21. Matches upper case letters; and matches lower case letters - Interactive (present letter tiles on screen; child matches pairs by clicking/dragging)
